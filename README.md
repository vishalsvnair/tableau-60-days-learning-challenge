# 📊 Tableau 60 Days Learning Challenge -- Interview Edition

Welcome to the **Tableau 60 Days Learning Challenge -- Interview
Edition**! 🚀

This repository is a structured learning roadmap designed to help
aspiring **Data Analysts**, **Business Intelligence Professionals**,
students, and working professionals master Tableau through consistent
daily practice.

Unlike traditional tutorials that focus only on features, this challenge
emphasizes **real-world business scenarios**, **hands-on learning**,
**interview preparation**, and **portfolio development**. Each day
introduces a new concept, explains why it matters, demonstrates how it's
applied in business, and provides practical exercises to reinforce your
understanding.

## 🎯 What You'll Learn

-   Connect Tableau to different data sources
-   Prepare, clean, and organize data
-   Build effective charts and visualizations
-   Create interactive dashboards
-   Master calculated fields, LOD expressions, and table calculations
-   Optimize dashboard performance
-   Solve real business problems using data
-   Prepare for Tableau interviews
-   Build portfolio-ready Tableau projects

## 📅 Learning Roadmap

  Week     Focus Area
